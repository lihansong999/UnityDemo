## 3D动作射击类网络游戏示例Demo

[在线运行 Demo(全屏效果更佳)](http://testtemp1.qujkj.com)

## 操作说明

1. WASD 控制移动方向，space 跳跃，按下 shift 加速移动
2. V 切换视角
3. T 打开菜单
4. 鼠标左键发射技能

## 项目结构

客户端工程在 URPStudy 目录，使用 Unity 2022.3.52f1c1 (URP)
服务端工程在 Server 目录，使用 Visual Studio 2022 C++
